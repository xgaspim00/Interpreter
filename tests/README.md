# IPP projekt – testy souladu šablon

V tomto adresáři se nachází skripty, které automaticky testují, zda se všechny tři šablony
chovají stejným způsobem:

- `int_correspondence` testuje deserializaci SOL-XML souborů do modelů,
- `tester_correspondence` testuje serializaci výstupních reportů do formátu JSON,
- `tester_cli_spec` testuje, zda se ve všech šablonách chová načítání parametrů příkazové
  řádky v souladu se zadáním.

Tyto testy nemusíte nijak řešit, jsou zde pro potřebu udržování šablon.

Všechen kód v tomto adresáři byl vygenerován s použitím OpenAI Codex.
